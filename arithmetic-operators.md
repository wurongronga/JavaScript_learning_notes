# arithmetic operators



| Operator | Description                                                          |
| -------- | -------------------------------------------------------------------- |
| +        | Addition                                                             |
| -        | Subtraction                                                          |
| \*       | Multiplication                                                       |
| \*\*     | Exponentiation ([ES2016](https://www.w3schools.com/js/js\_2016.asp)) |
| /        | Division                                                             |
| %        | Modulus (Division Remainder)                                         |
| ++       | Increment                                                            |
| --       | Decrement                                                            |
